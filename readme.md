# Dependencias Node:

1. Dentro da pasta "bin"
   `node init -y` //iniciar o node, criar package-lock
   `npm install ` // install dependencias
   `npm run clear` //limpar base de dados txt->json
   `npm start` // start servidor
   
2. Dentro da pasta "web"
   `npm install ` // install dependencias
   `npm start` // start servidor
